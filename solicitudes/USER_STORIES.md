# Historias de Usuario - Sistema de Solicitudes

## Alumno

### US-01: Crear Solicitud (Alumno)
**Descripción:**  
Como Alumno, quiero crear una nueva solicitud seleccionando el tipo y completando el formulario correspondiente, para iniciar un trámite.

### US-02: Visualizar Mis Solicitudes (Alumno)
**Descripción:**  
Como Alumno, quiero visualizar un listado de las solicitudes que he generado, para conocer su estado actual.

### US-03: Visualizar Seguimiento (Alumno)
**Descripción:**  
Como Alumno, quiero ver el detalle y el historial de seguimiento de mis solicitudes, para estar informado sobre el progreso y respuestas.

## Control Escolar

### US-04: Crear Tipos de Solicitud (Control Escolar)
**Descripción:**  
Como Control Escolar, quiero crear y gestionar los tipos de solicitudes disponibles en el catálogo, para definir los servicios ofrecidos.

### US-05: Crear Formularios de Solicitud (Control Escolar)
**Descripción:**  
Como Control Escolar, quiero crear formularios dinámicos para cada tipo de solicitud (campos y archivos), para asegurar la captura de información necesaria.

### US-06: Atender Solicitudes (Control Escolar)
**Descripción:**  
Como Control Escolar, quiero atender las solicitudes asignadas a mi área, registrando el seguimiento correspondiente para darles resolución.

### US-07: Crear Solicitud (Control Escolar)
**Descripción:**  
Como Control Escolar, quiero crear una nueva solicitud en el sistema, para iniciar un trámite propio o administrativo.

## Administrador

### US-17: Crear Tipos de Solicitud (Administrador)
**Descripción:**  
Como Administrador, quiero crear y gestionar los tipos de solicitudes disponibles en el catálogo.

### US-18: Crear Formularios de Solicitud (Administrador)
**Descripción:**  
Como Administrador, quiero crear formularios dinámicos para cada tipo de solicitud, definiendo campos y requisitos de archivos.

### US-19: Gestionar Usuarios y Roles
**Descripción:**  
Como Administrador, quiero gestionar el registro de usuarios y asignar sus roles correspondientes, para controlar el acceso al sistema.

### US-20: Dashboard de Métricas
**Descripción:**  
Como Administrador, quiero visualizar un dashboard con métricas clave (total de tickets, promedio de resolución, desglose por área) para monitorear el desempeño.

### US-21: Gráficas de Tendencia
**Descripción:**  
Como Administrador, quiero visualizar gráficas de tendencia de solicitudes por mes, semana y día, para identificar patrones.

### US-22: Exportación a PDF
**Descripción:**  
Como Administrador, quiero exportar la información de solicitudes y métricas a formato PDF, para generar reportes estáticos.

### US-23: Exportación a Excel
**Descripción:**  
Como Administrador, quiero exportar la información de solicitudes y métricas a formato Excel, para el análisis de datos.
